# ⚠️ CRITICAL POLICY: NO SYNTHETIC DATA

**Policy:** **NEVER insert synthetic, fake, or made-up data into the SETTLE database.**

---

## ✅ ALLOWED Data Sources

1. **Real Court Records**
   - Public court records from official sources
   - Verified case numbers and outcomes
   - Authentic settlement information

2. **Legal Publications**
   - Law blogs discussing real cases
   - Legal magazines with case studies
   - News articles about actual settlements
   - Must include source URL for verification

3. **Attorney Contributions**
   - Real cases from attorney clients
   - Properly anonymized
   - With consent and verification

---

## ❌ FORBIDDEN Data Sources

1. **Synthetic/Test Data**
   - Made-up case information
   - Research-based samples
   - Placeholder data
   - Test cases

2. **Estimated Data**
   - Guessed settlement amounts
   - Assumed case types
   - Inferred information without source

3. **Fake References**
   - Sample case numbers
   - Placeholder URLs
   - Test data markers

---

## 🔍 Verification Requirements

**Every case MUST have:**
- ✅ Real source URL (court record or publication)
- ✅ Verifiable case information
- ✅ Authentic settlement data
- ✅ No "SAMPLE" or "TEST" markers

**Before inserting ANY data:**
1. Verify source is real
2. Check source URL works
3. Confirm case information is authentic
4. Mark verification status

---

## 📝 Data Collection Process

1. **Collect from Real Sources Only**
   - Legal blogs/magazines
   - Court websites
   - Public records

2. **Extract Real Information**
   - Actual case details
   - Real settlement amounts
   - Authentic case types

3. **Verify Before Inserting**
   - Review source URL
   - Confirm authenticity
   - Check for synthetic markers

4. **Insert Only Verified Data**
   - Real cases only
   - Verified information
   - Authentic sources

---

## 🚫 Enforcement

**If synthetic data is detected:**
- ❌ Do NOT insert
- ❌ Remove immediately if found
- ❌ Report source
- ❌ Update collection process

**This policy is MANDATORY and NON-NEGOTIABLE.**
